# Real estate description service

### Technologies
- Back-end
  - Laravel
  - MySQL
- Front-end
  -  Vue.js(Admin panel)

### Features
- Parsing data about property from real estate sites(Zillow)
- Integration with ChatGPT
- Making subscription(Stripe)
- Coupons
- Tracking user activity
